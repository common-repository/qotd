Qotd simply displays random quotations.

== Description ==

This plugin for WordPress allows you to display random quotations on your weblog. 
The plugin supports several different quote 'providers' in different languages.
In this way your quotations should be unique even if others use the same plugin.
It's easy to setup and configure via an options panel.
The random quote is cached according with your frequency update preferences.

== Installation ==

1. Put the content of the zip in your plugins directory   
2. Activate the plugin
3. Configure your settings via the panel in Options
4. Add <?php Qotd(); ?> somewhere in your templates, or use the sidebar widget.
